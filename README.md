Here, I have uploaded the class works at 'IoT and Robotics' Course conducted by EDGE and supported by IIT,JU. I worked on: 

Microcontrollers:

Arduino Uno, Nano, Mega,
ESP8266, ESP32,
Raspberry PI zero, 2.


Sensors:

Fingerprint sensor,
Gas sensor,
Temperature sensor,
Humidity sensor,
Ultrasonic sensor,
Soil Moisture sensor,
LDR,
PIR sensor.


Modules:

GSM and GPRS,
LCD_i2c,
Keypad,
Servo motor,
Relay.


Applications used:

Blynk,
Arduino IDE.



Abbreviations:

EDGE = Enhancing Digital Government and Economy,
GPRS = General Packet Radio Service,
GSM = Global Systems for Mobile,
IDE = Integrated Development Environment,
IIT = Institute of Information and Technology,
I2C = Inter-Integrated Circuit,
JU = Jahangirnagar University,
LCD = Liquid Crystal Display,
LDR = Light Dependent Resistor,
PIR = Passive Infra-Red.
